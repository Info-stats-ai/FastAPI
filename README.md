A full-stack machine learning application that predicts health insurance premium categories using FastAPI backend and Streamlit frontend. The system uses a trained scikit-learn model to analyze user demographics, lifestyle factors, and location data to predict insurance premium categories.

## ‚ú® Features

### Backend (FastAPI)
- **ML-powered predictions**: Uses a trained scikit-learn model to predict insurance premium categories
- **Data validation**: Leverages Pydantic models with computed fields for robust input validation
- **Automatic calculations**: Automatically computes BMI, lifestyle risk, age groups, and city tiers
- **RESTful API**: Clean REST endpoints for easy integration
- **Interactive documentation**: Auto-generated Swagger UI and ReDoc documentation

### Frontend (Streamlit)
- **Interactive web interface**: User-friendly Streamlit app for easy predictions
- **Real-time BMI calculation**: Displays BMI and health status with visual indicators
- **Input validation**: Prevents invalid data entry with proper form controls
- **Beautiful UI**: Modern, responsive design with clear visual feedback
- **Error handling**: Comprehensive error messages and connection status
